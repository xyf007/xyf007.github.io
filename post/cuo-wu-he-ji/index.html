· 2020-11-04 · # 总结

错误合集

总结各种反复出现的错误。

  1. 忘记初始化（运行初始化函数、head 赋初值等）。
  2. 多测没清空。
  3. 没删调试语句。
  4. 多测的时候，读入一半就直接完成特判 continue，忘记读入该测试数据全部内容，导致输入错位。
  5. for (int i = n; i >= 1; i++)。
  6. for (int j = 1; j <= n; i++)。
  7. 有多个名字相近的函数的时候调用错误，如 Dfs()、Dfs2()。
  8. 各种边界情况忘记判断。
  9. 特判的时候写错。
    • 1ULL << 64 是会挂的。
    • 特判的答案也可能超过 264−12^{64}-1264−1。
  11. 变量名/函数名打错。
  12. Dijkstra 时堆中只存了编号。
  13. 快读没赋初值为 000。
  14. 滚动数组刷表法没清空上一轮。
  15. 多项式没清空数组。
  16. 长度不同的 FFT/NTT 没有重新计算 rev。
  17. 重载 cmp 的时候没加 const。
  18. 数组开小，有些维度可能是在值域上。
  19. 线段树/平衡树忘记 pushdown。
  20. 不满足二分性质的不要乱二分。
  21. long long 范围内两个数相除要用 long double。
  22. for (int j = 1; j <= n; j++) 写成 for (int j = 0; j < n; j++)。
  23. 数学题不要脑抽写模拟。
  24. 注意宏定义的优先级。
  25. 李超树 pushup 的时候不是赋值成儿子的信息。
  26. 即使理论复杂度正确也不要用 std::vector<char> 来判断两个字符串是否相等。
  27. 线段树没开 4 倍空间。
  28. 爆 long long 没有及时取 min。
  29. 用哈希判断字符串集合的时候没有把分隔符放进去。
  30. 树上背包合并没有清空数组。
